# Contributing Guide

Thank you for improving UPI Spam Detection.

## Development Setup

1. Install dependencies: `npm install`
2. Run app: `npm start`
3. Run quick syntax check: `npm run check`

## Contribution Workflow

1. Create a branch from `main`.
2. Keep each commit focused and descriptive.
3. Update documentation for user-visible behavior changes.
4. Validate app manually before opening PR.

## Good First Contributions

- Add scam patterns from new fraud campaigns.
- Improve explanatory UI messages and warnings.
- Expand multilingual keyword support.
- Add tests for scoring and false-positive handling.

## Pull Request Checklist

- [ ] Change is focused and documented.
- [ ] No unrelated files were modified.
- [ ] UI text is clear and actionable.
- [ ] Any new detection logic includes rationale.
