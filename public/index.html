UPI Fraud Keyword Detection System

Keyword-based scam message screening for payment and account security use cases.

Financial Security Context

UPI users routinely receive suspicious payment prompts, verification requests, and account warning messages. This application identifies risky terms in incoming text and helps users respond cautiously.

Detection Approach

The detector applies the Rabin-Karp algorithm to scan messages against a large fraud keyword dataset. Matching terms are highlighted and aggregated into a risk score.

Operational Use

This workflow can be integrated into support desks, fraud escalation teams, and customer-facing warning tools to flag potential social engineering attempts in real time.

System Capabilities

  • Authentication with local user management and session handling.
  • Protected detector access for authenticated users.
  • Rabin-Karp based string matching with hashing and sliding window logic.
  • Live keyword detection while typing and full analysis on demand.
  • Detailed algorithm explanation and interactive step visualization.
