# Security Policy

## Scope

This repository is an educational and screening tool for suspicious UPI message text.
It is not a full anti-fraud platform and should not be treated as the sole decision engine.

## Reporting Security Issues

If you discover a security issue in this repository:

1. Do not open a public issue with exploit details.
2. Share a private report with reproduction steps, impact, and mitigation suggestions.
3. Allow maintainers time to validate and patch before public disclosure.

## Data Handling Notes

- Demo authentication is client-side and uses local storage.
- Do not store real secrets or production credentials.
- Treat this app as a prototype/demo unless hardened for production use.
